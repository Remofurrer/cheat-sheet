Cheat Sheet

Tabellen Tags

Tag Name Beschreibung
<table> Tabelle Das Wrapper-Element für alle HTML-Tabellen
<thead> Tabellenkopf Die Reihe von Zeilen, die die Spaltenüberschriften in einer Tabelle definieren
<tbody> Tabellenkörper Der Satz von Zeilen, die tatsächliche Tabellendaten enthalten
<tfoot> Tabellenfuss Die Reihe von Zeilen, die die Fußzeile in einer Tabelle definieren
<th> Tabellenkopfinhalt Definiert eine Kopfzelle in einer Tabelle
<tr> Inhalt der Tabellenzeile Definiert die Zeilen in einer übergeordneten Zelle
<td> Tabelle Dateninhalt Definiert die Daten in einer Zeile
<caption> Definiert eine Tabellenbeschriftung Die Beschriftung für die Tabelle. Wie eine Figur an eine Figur
<colgroup> Definiert eine Gruppe von Spalten Gibt eine Gruppe von einer oder mehreren Spalten in einer Tabelle zur Formatierung an

Tabellen Attribute

Attribute erlaubt bei... Beschreibung
<colspan> th, td Erweitert eine Zelle auf eine Breite von 2 oder mehr Zellen
<rowspan> th, td Erweitert eine Zelle auf 2 oder mehr Zellen
<span> col Lässt die Spalte auf mehr als 2 oder mehr Spalten angewendet werden
<sortable> table Gibt an, dass die Tabelle sortiert werden soll. UPDATE: Mir wurde gesagt, dass dies wegen fehlender Implementierungen aus der Spezifikation entfernt wurde
<headers> td durch Leerzeichen getrennte Zeichenfolge, die den IDs der <th> für die Daten relevanten Elemente entspricht.
<scope> th row | col | rowgroup | colgroup (Standard) - Gibt im Wesentlichen die Achse der Kopfzeile an. Die Standardeinstellung ist, dass ein Header eine Spalte leitet, was typisch ist. Eine Zeile kann jedoch auch mit einem Header beginnen, in dem Sie diesen Header auf die Zeile oder Zeilengruppe beschränken würden.

Veraltete Attribute

Veraltete Attribute Was stattdessen zu verwenden
align Verwenden Sie stattdessen die Eigenschaft "float"
valign Verwenden Sie stattdessen die Eigenschaft "vertical-align"
char Die richtige Antwort ist die Verwendung von text-align: "x"; Dabei ist x das Zeichen, an dem ausgerichtet werden soll, aber es ist noch nirgendwo implementiert. Dieses Attribut wird jedoch auch nicht unterstützt, sodass kein großer Verlust entsteht
charoff Siehe oben
bgcolor Verwenden Sie stattdessen die Eigenschaft "background-color"
abbr Erwägen Sie, den Zelleninhalt mit einem unabhängigen abgekürzten Inhalt selbst zu beginnen, oder verwenden Sie den abgekürzten Inhalt als Zelleninhalt und verwenden Sie den langen Inhalt als Beschreibung der Zelle, indem Sie ihn in das title-Attribut einfügen
axis Verwenden Sie stattdessen die Eigenschaft "scope"
border Verwenden Sie stattdessen die Eigenschaft "border"
cellpadding Verwenden Sie stattdessen die Eigenschaft "padding"
cellspacing Verwenden Sie stattdessen die Eigenschaft "border-spacing"
frame Verwenden Sie stattdessen die Eigenschaft "border"
rules Verwenden Sie stattdessen die Eigenschaft "border"
summary Verwenden Sie stattdessen die Eigenschaft <caption>
width Verwenden Sie stattdessen die Eigenschaft "width"
